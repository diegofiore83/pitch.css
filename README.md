# pitch.css

## Pure HTML/CSS football pitch - Responsive Desing

This is a simple HTML/CSS set of rules to design a responsive football pitch (vertical).

## Getting Started

The main folder contains the necessary files in SASS format.

- npm install
- npm start
- open index.html
